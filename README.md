Error 404: Kaapi Not Found Café
A responsive front-end website for a café brand inspired by developer culture, designed and implemented using standard web technologies.
The project emphasizes clean UI design, usability, and responsive behavior across devices.

Project Description:
Error 404: Kaapi Not Found Café is a static website developed to represent a fictional café with a technology-themed identity.
The application demonstrates core front-end development principles including semantic HTML structure, responsive CSS styling, and interactive JavaScript components.
The project is suitable for academic submission, portfolio presentation, and front-end skill demonstration.

Key Features:
Responsive layout compatible with desktop, tablet, and mobile screens
Clean and structured UI with consistent typography and spacing
Interactive elements implemented using JavaScript
Organized and maintainable project structure
Cross-browser compatibility

Technologies Used:

CSS3	                          Layout, styling, responsiveness
JavaScript	                    Client-side interactivity
